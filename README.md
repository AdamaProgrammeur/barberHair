# Barber Hair – Gestion de Salon

**Barber Hair** est une application web Django pour gérer un salon de coiffure.  
Elle permet de gérer les clients, les services, la file d’attente, les paiements, et propose un **dashboard administrateur**.

---

## 🛠️ Fonctionnalités

- Gestion des clients (ajout, modification, suppression)  
- Gestion des services proposés  
- Gestion de la file d’attente des clients  
- Gestion des paiements (total et paiement du jour)  
- Dashboard personnalisé pour suivre l’activité du salon  
- Authentification utilisateur et superadmin  
- Configuration multi-utilisateurs (admin / employé)  
- Interface simple avec sidebar et logo
